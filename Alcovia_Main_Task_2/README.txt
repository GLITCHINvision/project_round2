
# Alcovia Main Task Submission

## Description
This project implements the Personalized Learning Path feature as per the requirements.

### Deliverables
1. `database_schema.json`: Contains the mock database schema for storing student information.
2. `recommendation_algorithm.py`: Python script implementing the recommendation algorithm.
3. `interface_mockup.json`: JSON output simulating the frontend interface.

### Instructions to Run
1. Open `recommendation_algorithm.py` in any Python IDE or editor.
2. Run the script to see the recommendations for the mock student.

### Notes
- The recommendations are mock-generated based on predefined interests and preferences.
